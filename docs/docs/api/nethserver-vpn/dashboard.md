# dashboard

Read OpenVPN and IPSec status.

## read

The read API takes an `action` field.

Valid actions are:

- `status`: return configuration status
- `chart`: return chart for the given interface

### Input

#### status

Input example:
```json
{
  "action": "status"
}
```

#### chart

The API requires an extra `name` field which contain the interface name.

Input example:
```json
{
  "action": "chart",
  "name": "tunrw"
}
```

### Output

#### status

Return current IPSec tunnels and OpenVPN status.

Output example:
```json
{

    "tunnels": {
      "interfaces": [
        "tunctun1",
        "tunctun2",
        "tuntun1",

      "connected": 1,
      "total": 4
    },
    "roadwarrior": {
      "auth": "certificate",
      "interfaces": [
        "tunrw"
      ],
      "mode": "routed",
      "status": "enabled",
      "connected": 0,
      "port": "1194",
      "total": 4
    }
  },
  "ipsec": {
    "connected": 0,
    "total": 2
  }
}
```
